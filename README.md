# 🌤️ Clima App

Weather application built with Angular and Open-Meteo API.

![Status](https://img.shields.io/badge/Status-Active-green)

## Demo

Live at: **https://dalex188.github.io/AppClima**

## Features

- Real-time weather data
- 7-day forecast
- Dynamic theme by weather condition
- Dark/Light mode toggle
- Responsive design
- Free API - no API key required

## Tech Stack

| Technology | Purpose |
|------------|--------|
| [Angular](https://angular.io/) | Framework |
| [Open-Meteo API](https://open-meteo.com/) | Weather data (free) |
| [Font Awesome](https://fontawesome.com/) | Icons |

## Development

```bash
# Install dependencies
npm install

# Run locally
ng serve

# Build for production
ng build
```

## Deploy

Build output is in `dist/clima-app/browser/`. GitHub Pages serves from `/docs`.
